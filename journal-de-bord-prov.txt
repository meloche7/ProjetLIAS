Le but est de créer un widget pour le site web du lias et de récupérer les données d'un post/page instagram grâce au webscrapping

Le site fonctionne grâce à jekyll qui est un générateur de site web statique, cela veut dire qu'on ne conserve rien,
aucune données est conservée sur le site web. La synchronisation du site web s'effectue grâce à jenkins lorsqu'on effectue
un push sur git le site se met à jour, il se met aussi à jour automatiquement toute les heures.
